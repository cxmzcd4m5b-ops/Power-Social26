## 📝 SETUP (One-Time)

1. Get key: https://platform.openai.com/api-keys
2. Open `.env` file
3. Add: `OPENAI_API_KEY=sk-your-key`
4. Save and refresh browser

## 🎯 HOW TO USE

### Upload → Generate → Copy → Post

1. **Upload** videos/photos (drag & drop)
2. **Click** "Generate Social Media Content"
3. **Wait** 20-45 seconds
4. **Copy** captions for each platform
5. **Post** to social media!

## ❌ TROUBLESHOOTING

### "Failed to generate"
→ Check API key in `.env`
→ Refresh browser

### Files won't upload
→ Check format (MP4, JPG, PNG)
→ Check size (<50MB)
→ Try fewer files

### Server won't start
→ Port 3000 in use?
→ Run `npm install`
→ Restart terminal
