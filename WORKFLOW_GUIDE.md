# 🎬 Power Social - Complete Workflow Guide

## From Log Cabin Site → Viral Social Media Post

This guide walks you through the entire process from capturing content on-site to posting on social media.

---

## 📱 STEP 1: Capture Content On-Site

### What to Film:

#### 🎥 Essential Shots:
1. **Progress Overview** (Wide shot)
   - Show overall cabin progress
   - Film from same angle each day for time-lapse potential

2. **Close-Up Techniques** (Detail shots)
   - Hand-fitting logs
   - Notching techniques
   - Tool usage
   - Traditional methods

3. **Satisfying Moments** (Action shots)
   - Log sliding into place
   - Perfect fits
   - Clean cuts
   - Final placements

4. **Before/After** (Comparison shots)
   - Start of day vs end of day
   - Raw materials → finished section
   - Problem → solution

#### 📸 Photo Opportunities:
- Dramatic angles
- Detail shots of craftsmanship
- You at work (for authenticity)
- Materials and tools
- Scenic location shots

### Recording Tips:
✅ **Always shoot vertical (9:16)** for social media
✅ Hold phone steady (use tripod if possible)
✅ Good lighting (golden hour = best)
✅ Clean lens
✅ Record 10-20 second clips
✅ Capture multiple angles

---

## 💻 STEP 2: Transfer Files to Computer

### Method A: Direct Cable
1. Connect phone to computer via USB
2. Open file explorer
3. Navigate to phone → DCIM → Camera
4. Copy videos/photos to computer

### Method B: Cloud Storage
1. Upload to Google Photos / iCloud
2. Download to computer

### Method C: AirDrop (Mac)
1. Select files on iPhone
2. AirDrop to Mac
3. Save to folder

**Organize**: Create folders by date or project phase

---

## 🖥️ STEP 3: Open Power Social

### Launch App:
1. Open terminal/command prompt
2. Navigate to project: `cd "Power Social"`
3. Start server: `npm run dev`
4. Open browser: http://localhost:3000

**OR** - If already running, just open the browser!

---

## 📤 STEP 4: Upload Content

### Upload Process:

1. **Drag & Drop Method**:
   - Open Power Social in browser
   - Drag video/photo files from folder
   - Drop onto upload area
   - See files appear as thumbnails

2. **Browse Method**:
   - Click on upload area
   - File picker opens
   - Select one or multiple files
   - Click "Open"

### What You'll See:
- ✅ File preview (video player or image)
- ✅ File name
- ✅ File type icon
- ✅ Remove button (if needed)

**Pro Tip**: Upload 3-5 clips at once for better AI context

---

## ✨ STEP 5: Generate Content

### Click the Big Button:
**"Generate Social Media Content"** (orange/amber button)

### What Happens:
1. **Uploading** (2-5 seconds)
   - Files sent to server

2. **AI Analyzing** (10-20 seconds)
   - GPT-4 examines your content
   - Identifies scenes and materials
   - Detects key moments
   - Understands construction phase

3. **Content Creation** (10-20 seconds)
   - Generates captions for each platform
   - Creates trending hashtags
   - Suggests music
   - Provides editing tips

**Total Time**: 20-45 seconds

### Progress Indicator:
You'll see: "Analyzing & Creating Content..." with spinning icon

---

## 📋 STEP 6: Review Generated Content

### You'll Get 4 Platform Cards:

#### 🎵 TikTok Card
- **Caption**: Viral hook + engaging story
- **Hashtags**: 15 trending tags
- **Music**: 3 current trending tracks
- **Edit Tips**: Timing, cuts, effects

#### 📸 Instagram Card
- **Caption**: Aesthetic, story-driven
- **Hashtags**: Mix of popular + niche
- **Music**: Reels-friendly tracks
- **Edit Tips**: Visual effects, transitions

#### 👥 Facebook Card
- **Caption**: Longer, storytelling format
- **Hashtags**: Community-focused
- **Music**: Background ambiance
- **Edit Tips**: Engagement strategies

#### ▶️ YouTube Shorts Card
- **Caption**: SEO-optimized title
- **Hashtags**: Searchable tags
- **Music**: Copyright-safe options
- **Edit Tips**: Tutorial format

---

## 📝 STEP 7: Copy Content

### For Each Platform:

1. **Review** the generated content
2. **Click** "Copy Caption & Hashtags" button
3. **Confirmation**: Button turns green "✓ Copied!"
4. Content is now in your clipboard

**What Gets Copied**:
```
[Caption text]

[All hashtags]
```

Ready to paste directly into social media!

---

## ✂️ STEP 8: Edit Your Video

### Follow the AI's Video Edit Tips:

#### Example TikTok Edits:
1. **Open** CapCut or preferred editor
2. **Import** your video
3. **Trim** to 15-60 seconds
4. **Add** text overlay from AI suggestions
5. **Speed up** repetitive sections (1.5-2x)
6. **Add** suggested music
7. **Export** in 9:16 vertical format

#### Key Edits (from AI tips):
- Hook in first 3 seconds
- Text overlays at key moments
- Speed ramping for impact
- Smooth transitions
- End with CTA (call to action)

### Editing Tools:
- **CapCut** (Free, easy) - Recommended!
- **iMovie** (Mac, free)
- **Adobe Premiere Rush** (Simple, paid)
- **DaVinci Resolve** (Advanced, free)

---

## 📱 STEP 9: Post to Social Media

### For Each Platform:

#### 🎵 TikTok:
1. Open TikTok app
2. Tap "+" to create
3. Upload edited video
4. **Paste** caption (from Power Social)
5. **Add** suggested music manually
6. Adjust covers and settings
7. **Post** → click share

#### 📸 Instagram:
1. Open Instagram app
2. Tap "+" → Reel
3. Upload edited video
4. **Paste** caption
5. **Add** music from suggestions
6. Add cover image
7. Share to feed + stories

#### 👥 Facebook:
1. Open Facebook
2. Create post
3. Upload video
4. **Paste** caption and hashtags
5. Tag location if relevant
6. **Post**

#### ▶️ YouTube:
1. Open YouTube Studio
2. Create → Upload videos
3. Select video
4. **Paste** title (from caption)
5. Add description with hashtags
6. Select "Shorts" if vertical
7. **Publish**

### Morning (On Site):
- **7:00 AM**: Capture "start of day" shots
- **8:00-12:00 PM**: Record interesting moments as they happen
- **12:00 PM**: Lunch break - quick time-lapse setup

### Afternoon (On Site):
- **1:00-5:00 PM**: Capture progress and techniques
- **5:00 PM**: Final "end of day" comparison shot

### Evening (Home):
- **6:00 PM**: Transfer files to computer
- **6:15 PM**: Upload to Power Social
- **6:20 PM**: Generate content (takes ~30 seconds)
- **6:30 PM**: Edit video with AI tips
- **7:00 PM**: Post to all platforms
- **7:30 PM**: Engage with comments

**Total time**: ~90 minutes from capture to post

## 💡 PRO WORKFLOW TIPS

### Batch Processing:
Instead of daily, try:
1. **Film** all week (Monday-Friday)
2. **Weekend**: Process all content at once
3. **Generate** content for 5-7 posts
4. **Edit** all videos in one session
5. **Schedule** posts for following week

### Time Savers:
✅ Film everything vertical (9:16) from start
✅ Use same music library as AI suggestions
✅ Create video editing templates
✅ Save hashtag sets by project phase
✅ Batch similar content together

### Quality Boosters:
✅ Film during golden hour (sunrise/sunset)
✅ Clean phone lens before every shot
✅ Use phone tripod for steady shots
✅ Capture ambient sounds (sawing, chopping)
✅ Get establishing shots of location

## 🔧 TROUBLESHOOTING WORKFLOW

### If Content Doesn't Perform:

1. **Review** AI suggestions again
2. **Check** if you followed edit tips
3. **Verify** you used suggested music
4. **Ensure** hashtags are current
5. **Try** different posting times
6. **Re-generate** with new clips
7. **Test** variations

### If Upload Fails:

1. **Check** file size (< 50MB)
2. **Compress** video if needed
3. **Verify** file format (MP4, MOV, JPG, PNG)
4. **Try** fewer files at once
5. **Refresh** browser
6. **Restart** dev server if needed
